== Description ==

This plugin is to be used with the Coworks app to give your users a place to submit a request form whether it be a membership request or a tour request, or other types.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Activate the plugin through the 'Plugins' screen in WordPress
2. Go to the page editor and select the WSYWIG editor and find the button labeled "Coworks".
3. Click this and enter your subdomain and form type.
